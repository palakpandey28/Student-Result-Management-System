---------------PROJECT TITLE-------------------

 * Student Result Management System * 

------------overview of project----------------

The Student Result Management System is a simple Java-based console application that 
allows entering student details, calculating total & average marks, assigning grades, sorting results, and
displaying the class topper.

--------------Features------------------------

1. Enter multiple student records

2. Calculates Total Marks

3. Calculates Average Marks

4. Automatically assigns Grade (A/B/C)

5. Sorts students by Total Marks

6. Displays the Topper

7 Clean and simple console output


-------------- Technologies Used--------------------

1. Java

2. VS Code (with Java extensions)

3. Console Output

---------------Steps to Run the Project-----------------

You do NOT need to use terminal or compile manually.
Just follow these steps:

1. Open the project folder in VS Code

2. Open the file StudentResultManagement.java

3. Click the Run ▶ button at the top

4. The program will start and ask for inputs in the terminal panel of VS Code


-------------Instructions for Testing-------------------

1. Run the program

2. Enter number of students

3. Enter each student’s name, roll number, class, and marks

Program will display:

          a. Sorted Result Table

          b. Topper Details
